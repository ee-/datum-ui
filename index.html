Foundation

Design Identity

01

Clarity over Clutter

Every element earns its space. Data density with functional whitespace.

02

Geometric Precision

Sharp edges, chamfered corners, deliberate alignment. No organic curves.

03

Industrial Contrast

Mustard accents against charcoal on off-white. Small, intentional palette.

04

Flat & Borderline

No drop shadows. Depth via crisp 1px borders and background tints only.

Do / Don't

Design guardrails. Follow these to stay on-brand.

✓

Do

Use chamfered clip-path on cards and buttons. This is the signature shape of the system.

Use font-mono for data. Numbers, IPs, timestamps, and technical identifiers should always be monospaced.

Use flat 1px borders for depth. Never use box-shadow on cards. The only exception is tooltips.

Uppercase all headings with tight tracking. This creates the industrial feel.

Use leading-relaxed for body text to ensure readability on dense pages.

✗

Don't

No rounded corners. Never use rounded-lg, rounded-full, or border-radius. The chamfer IS the shape language.

No box-shadows on cards. This system is deliberately flat. Use border-border for separation.

No default Tailwind colors. Never use bg-blue-500, text-red-500 etc. Use only the named tokens.

No bouncy animations. Transitions are mechanical — ease-standard, 150ms. No spring or bounce.

No gradients on surfaces. Backgrounds are flat solid fills. The dot grid is the only texture.
